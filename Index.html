What is EcoMon™?

EcoMon™ is a open system for collecting, gathering and analysing information on emissons at Truro College. Version 1.0 (Current Version) focuses on displaying information however analytical tools are planned for future development.

How do I get involved?

By registering with us, you can volunteer to collect information about how classrooms use electricity to contribute in an effort to reduce greenhouse emissons.

There can be many reasons why a user can't log on, eg: problems with the username, password, user account settings or security settings. Some of the main causes of logon problems are as follows:

  • Incorrect username: check that the username is correct by using Active Directory Users And Computers to verify that the name is spelled correctly.

  • Incorrect password: check that the password was entered in the proper case (and that the Caps Lock key isn't on), the password hasn't expired and the account has not been locked out. If all else fails, you can assign a new password through Active Directory Users And Computers.

  • Prohibitive user rights: does the user have permission to log on locally at the computer? Note that regular users do not have normally have permission to log on locally at the domain controller, as it is assumed that they will log on to the domain from network workstations.

  • Disabled or deleted account: check whether an account has been disabled or deleted by examining the account properties via Active Directory Users And Computers.

  • Unavailable domain controller, DNS Server or Global Catalog: is the domain controller available to authenticate the user's request? If the domain controller is down, the user will not be able to log on until it comes back up. A DNS server and the Global Catalog (if the Active Directory has multiple domains) are also required.
